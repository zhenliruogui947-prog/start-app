# start-app — 目覚まし時計アプリ

## プロジェクト概要

シンプルな目覚まし時計アプリ。ブラウザだけで動作し、サーバー不要。

## 技術スタック

| 技術 | 役割 |
|------|------|
| HTML | 画面構造（時刻表示、入力フォーム、ボタン） |
| CSS | レイアウト・スタイル・アニメーション |
| JavaScript | ロジック全般（時刻取得・アラーム・音声） |

外部ライブラリ・フレームワーク：なし（バニラ JS のみ）

## ファイル構成

```
start-app/
├── index.html       # メイン画面
├── style.css        # スタイル
├── script.js        # アプリロジック
└── alarm.mp3        # アラーム音（任意）
```

## 主な機能

- 現在時刻のリアルタイム表示（`setInterval` で毎秒更新）
- アラーム時刻の設定・解除
- アラーム音の再生（Web Audio API または `<audio>` タグ）
- アラーム設定の保存（`localStorage`）

## 開発ルール

- サーバー不要：ファイルをブラウザで直接開いて動作確認する
- 日本語 UI：ラベルやメッセージはすべて日本語
- コメントは最小限：自明なコードにコメントを書かない
- モバイル対応：スマートフォンでも使えるレスポンシブデザイン

## 動作確認方法

```
index.html をブラウザにドラッグ＆ドロップして開く
```
