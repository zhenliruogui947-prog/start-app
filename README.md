# ⏰ スタートアプリ

出発時刻から逆算して、最適な起床時刻を提案する目覚まし時計アプリです。
ブラウザだけで動作し、インストール不要で使えます。

🌐 **公開URL**: https://zhenliruogui947-prog.github.io/start-app/

---

## 主な機能

### ⏱ 起床時刻の自動計算
出発時刻・準備時間・駅までの徒歩時間を入力するだけで、起床時刻を逆算します。
電車の発車時刻を登録すると、乗るべき電車を自動選択して最適な時刻を算出します。

### ☔ 天気連携
- Open-Meteo API（無料・APIキー不要）で**起床時刻の時間帯**の天気予報を取得
- 雨・雪・霧などの悪天候時は自動で **15分早起き** に調整
- 都市名で場所を登録（市・区レベルまで対応）

### 🔔 アラーム
- 推奨起床時刻で自動セット、または手動で時刻を指定
- アラーム音を3種類から選択

| 音の種類 | 特徴 |
|----------|------|
| 📣 さわやか | 鳥のさえずり音（周波数グライドで自然な音色を再現） |
| 🎵 うきうき | ド・ミ・ソ・ド の上昇メロディ |
| 🔔 がんばるぞ！ | 激しいベル音の4連打 |

### 💤 スヌーズ
- 3分間隔、最大5回まで

### 🎋 おみくじ
アラームを止めると今日の運勢を表示（ON/OFF 切替可）

| 運勢 | |
|------|-|
| 🌟 最高の1日 | 😊 うれしい1日 |
| 🌸 平和な1日 | 💨 忙しい1日 |
| ⚠️ 注意が必要な日 | |

同じ日は何度引いても同じ運勢が出ます。

### 📝 メモ
アラームを止めた後に表示されるメモ機能。
今日やること・持ち物などを前日に書いておけます。

## 使い方

1. [アプリを開く](https://zhenliruogui947-prog.github.io/start-app/)
2. 天気カードの「📍 変更」から都市を登録
3. 出発時刻・準備時間・駅までの時間を入力
4. 「⏱ 起床時刻を計算する」を押す
5. 「推奨時刻でセット」でアラームをセット

### ローカルで動かす場合
```
index.html をブラウザにドラッグ＆ドロップして開く
```
サーバー不要で動作します。

## 技術スタック

| 技術 | 役割 |
|------|------|
| HTML / CSS / JavaScript | アプリ全体（外部ライブラリなし） |
| Web Audio API | アラーム音の生成 |
| Geolocation API | 位置情報の取得（使用しない設定も可） |
| localStorage | 設定の保存 |

### 使用 API

| API | 用途 | 費用 |
|-----|------|------|
| [Open-Meteo](https://open-meteo.com/) | 天気予報（時間別） | 無料 |
| [Nominatim / OpenStreetMap](https://nominatim.org/) | 都市名の検索・座標変換 | 無料 |
